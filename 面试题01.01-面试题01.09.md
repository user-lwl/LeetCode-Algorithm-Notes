# 面试题 01.01.判定字符是否唯一

```java
class Solution {
    public boolean isUnique(String astr) {
        char[] sChat = astr.toCharArray();
        Set<Character> tempSet = new HashSet<>();
        for (int i = 0; i < sChat.length; i++) {
            Character tempChar = sChat[i];
            if (!tempSet.add(tempChar)) {
                return false;
            }
        }
        return true;
    }
}
```

# 面试题 01.02.判定是否互为字符重排

```java
class Solution {
    public boolean CheckPermutation(String s1, String s2) {
        char[] s1Char = s1.toCharArray();
        char[] s2Char = s2.toCharArray();
        if (s1Char.length != s2Char.length) {
            return false;
        }
        Map<Character, Integer> tempMap = new HashMap<>();
        for (int i = 0; i < s1Char.length; i++) {
            tempMap.put(s1Char[i], tempMap.getOrDefault(s1Char[i], 0) + 1);
        }
        for (int i = 0; i < s2Char.length; i++) {
            int cnt = tempMap.getOrDefault(s2Char[i], 0);
            if (cnt == 0) {
                return false;
            }
            tempMap.put(s2Char[i], cnt - 1);
        }
        return true;
    }
}
```
