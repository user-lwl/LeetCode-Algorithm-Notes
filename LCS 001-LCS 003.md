# LCS 01.下载插件

```java
class Solution {
    public int leastMinutes(int n) {
        int size = 1;
        int minute = 0;
        while (size < n) {
            size *= 2;
            minute++;
        }
        return minute + 1;
    }
}
```

# LCS 02.完成一半的题目

```java
class Solution {
    public int halfQuestions(int[] questions) {
        Integer[] tempList = new Integer[1001];
        for (int i = 0; i < tempList.length; i++) {
            tempList[i] = 0;
        }
        int problem = questions.length;
        int person = problem / 2;
        for (int i = 0; i < questions.length; i++) {
            tempList[questions[i]]++;
        }
        Arrays.sort(tempList, Collections.reverseOrder());
        int cnt = 0;
        for (int i = 0; i < tempList.length; i++) {
            int problemNumber = tempList[i];
            if (problemNumber >= person) {
                return cnt + 1;
            }
            cnt++;
            person -= problemNumber;
        }
        return cnt;
    }
}
```
