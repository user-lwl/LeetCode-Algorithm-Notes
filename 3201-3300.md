# 3216.交换后字典序最小的字符串

```java
class Solution {
    public String getSmallestString(String s) {
        char[] sChar = s.toCharArray();
        for (int i = 0; i < sChar.length - 1; i++) {
            if (sChar[i] > sChar[i + 1] && sChar[i] % 2 == sChar[i + 1] % 2) {
                char temp = sChar[i];
                sChar[i] = sChar[i + 1];
                sChar[i + 1] = temp;
                break;
            }
        }
        return new String(sChar);
    }
}
```
