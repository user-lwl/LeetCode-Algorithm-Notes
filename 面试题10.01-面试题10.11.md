# 面试题 10.01.合并排序的数组

```java
class Solution {
    public void merge(int[] A, int m, int[] B, int n) {
        int indexA = m - 1;
        int indexB = n - 1;
        int indexEnd = A.length - 1;
        while(indexA >= 0 && indexB >= 0) {
            if (A[indexA] > B[indexB]) {
                A[indexEnd--] = A[indexA--];
            } else {
                A[indexEnd--] = B[indexB--];
            }
        }
        while (indexA >= 0) {
            A[indexEnd--] = A[indexA--];
        }
        while (indexB >= 0) {
            A[indexEnd--] = B[indexB--];
        }
    }
}
```
