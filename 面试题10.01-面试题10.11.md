# 面试题 10.01.合并排序的数组

```java
class Solution {
    public void merge(int[] A, int m, int[] B, int n) {
        int indexA = m - 1;
        int indexB = n - 1;
        int indexEnd = A.length - 1;
        while(indexA >= 0 && indexB >= 0) {
            if (A[indexA] > B[indexB]) {
                A[indexEnd--] = A[indexA--];
            } else {
                A[indexEnd--] = B[indexB--];
            }
        }
        while (indexA >= 0) {
            A[indexEnd--] = A[indexA--];
        }
        while (indexB >= 0) {
            A[indexEnd--] = B[indexB--];
        }
    }
}
```

# 面试题 10.02.变位词组

```java
class Solution {
    public List<List<String>> groupAnagrams(String[] strs) {
        Map<String, List<String>> tempMap = new HashMap<>();
        for (int i = 0; i < strs.length; i++) {
            int[] tempInt = new int[26];
            for (int j = 0; j < strs[i].length(); j++) {
                tempInt[strs[i].charAt(j) - 'a']++;
            }
            String key = Arrays.toString(tempInt);
            List<String> tempList = tempMap.getOrDefault(key, null);
            if (tempList == null) {
                tempList = new ArrayList<>();
                tempMap.put(key, tempList);
            }
            tempList.add(strs[i]);
        }
        List<List<String>> resultList = new ArrayList<>();
        for (String key : tempMap.keySet()) {
            resultList.add(tempMap.get(key));
        }
        return resultList;
    }
}
```



# 面试题 10.05.稀疏数组搜索

```java
class Solution {
    public int findString(String[] words, String s) {
        for (int i = 0; i < words.length; i++) {
            if (s.equals(words[i])) {
                return i;
            }
        }
        return -1;
    }
}
```
