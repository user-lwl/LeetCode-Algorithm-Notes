# 2706.购买两块巧克力

```java
class Solution {
    public int buyChoco(int[] prices, int money) {
        Arrays.sort(prices);
        if (prices[0] + prices[1] <= money) {
            return money - prices[0] - prices[1];
        }
        return money;
    }
}
```



# 2730.找到最长的半重复子字符串

```java
class Solution {
    public int longestSemiRepetitiveSubstring(String s) {
        int left = 0;
        int right = 0;
        int fixCount = 0;
        int max = 0;
        char[] c = s.toCharArray();
        int len = c.length;
        while (right < len) {
            if (right > 0 && c[right] == c[right - 1]) {
                fixCount++;
            }
            while (fixCount > 1) {
                left++;
                if (left > 0 && c[left] == c[left - 1]) {
                    fixCount--;
                }
            }
            max = Math.max(max, right - left + 1);
            right++;
        }
        return max;
    }
}
```

# 2733.既不是最小值也不是最大值

```java
class Solution {
    public int findNonMinOrMax(int[] nums) {
        int len = nums.length;
        int[] cnt = new int[101];
        for (int i = 0; i < len; i++) {
            cnt[nums[i]]++;
        }
        int temp = 0;
        int count = 2;
        for (int i = 0; i < cnt.length; i++) {
            if (cnt[i] != 0) {
                if (temp != 0 && count == 0) {
                    return temp;
                }
                temp = i;
                count--;
            }
        }
        return -1;
    }
}
```

# 2784.检查数组是否是好的

```java
class Solution {
    public boolean isGood(int[] nums) {
        Arrays.sort(nums);
        if (nums.length == 1) {
            return false;
        }
        for (int i = 0; i < nums.length; i++) {
            if (i == nums.length - 1) {
                if (nums[i] != nums[i - 1]) {
                    return false;
                }
            } else {
                if (nums[i] != i + 1) {
                    return false;
                }
            }
        }
        return true;
    }
}
```



# 2789.合并后数组中的最大元素

```java
class Solution {
    public long maxArrayValue(int[] nums) {
        long sum = nums[nums.length - 1];
        for (int i = nums.length - 2; i >= 0; i--) {
            sum = nums[i] <= sum ? nums[i] + sum : nums[i];
        }
        return sum;
    }
}
```
