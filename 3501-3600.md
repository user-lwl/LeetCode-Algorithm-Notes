# 3536.两个数字的最大乘积

```java
class Solution {
    public int maxProduct(int n) {
        char[] nChar = String.valueOf(n).toCharArray();
        Arrays.sort(nChar);
        int number1 = nChar[nChar.length - 2] - '0';
        int number2 = nChar[nChar.length - 1] - '0';
        return number1 * number2;
    }
}
```

# 3545.不同字符数量最多为K时的最少删除数

```java
class Solution {
    public int minDeletion(String s, int k) {
        char[] sChar = s.toCharArray();
        int[] tempList = new int[26];
        for (int i = 0; i < sChar.length; i++) {
            tempList[sChar[i] - 'a']++;
        }
        Arrays.sort(tempList);
        int result = 0;
        for (int i = 0; i < tempList.length - k; i++) {
            result += tempList[i];
        }
        return result;
    }
}
```
