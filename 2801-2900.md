# 2824.统计和小于目标的下标对数目

```java
class Solution {
    public int countPairs(List<Integer> nums, int target) {
        Collections.sort(nums);
        int result = 0;
        for (int i = 0; i < nums.size(); i++) {
            int temp1 = nums.get(i);
            for (int j = i + 1; j < nums.size(); j++) {
                int temp2 = nums.get(j);
                if (temp1 + temp2 < target) {
                    result++;
                } else {
                    break;
                }
            }
        }
        return result;
    }
}
```

# 2864.最大二进制奇数

```java
class Solution {
    public String maximumOddBinaryNumber(String s) {
        char[] c = s.toCharArray();
        int num = 0;
        for (int i = 0; i < c.length; i++) {
            if (c[i] == '1') {
                num++;
            }
        }
        String result = "";
        for (int i = 0; i < c.length; i++) {
            if (i == c.length - 1) {
                result += "1";
                return result;
            }
            if (num > 1) {
                result += "1";
                num--;
            } else {
                result += "0";
            }
        }
        return result;
    }
}
```
